Politique de Confidentialité - BlindBeat

Dernière mise à jour : 23 mai 2024

Bienvenue sur BlindBeat. La protection de votre vie privée est une priorité pour nous. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application mobile.

1. Collecte des Données

Nous collectons les types d'informations suivants pour assurer le bon fonctionnement de BlindBeat :

  • Informations fournies par l'utilisateur : Votre pseudonyme (pseudo) choisi lors de l'accès à l'application.
  • Données techniques : Type d'appareil, version du système d'exploitation, et identifiants uniques de l'appareil (IDFV/Android ID).
  • Données d'utilisation : Statistiques de jeu (scores, victoires, thèmes préférés) pour votre profil local.

2. Services Tiers

BlindBeat utilise des services tiers qui peuvent collecter des données conformément à leurs propres politiques :

  • Google AdMob : Utilisé pour diffuser des publicités. AdMob peut collecter des identifiants publicitaires (AAID/IDFA) pour personnaliser les annonces.
  • Firebase (Google) : Utilisé pour les fonctionnalités multijoueurs en temps réel et les rapports d'erreurs (Crashlytics).
  • API Deezer : Utilisée pour diffuser les extraits musicaux. Aucune donnée personnelle n'est partagée avec Deezer par l'application.

3. Permissions Requises

L'application sollicite les permissions suivantes :

  • Internet & État du Réseau : Pour le jeu en ligne et le streaming musical.

4. Conservation des Données

Les données liées à votre profil (pseudo et statistiques) sont stockées soit localement sur votre appareil, soit de manière sécurisée via Firebase. Vous pouvez demander la suppression de votre pseudo en nous contactant.

5. Sécurité

Nous mettons en œuvre des mesures de sécurité standard pour protéger vos informations contre tout accès non autorisé ou toute modification.

6. Vos Droits

Conformément au RGPD, vous disposez d'un droit d'accès, de rectification et de suppression de vos données. Pour toute demande, veuillez nous contacter par e-mail.
